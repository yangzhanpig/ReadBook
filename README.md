# 嘉泽小说阅读器

    一直爱看小说，苦于各种小说广告，所以利用业余时间写了这个软件，目前为止只实现了部分功能，后续逐渐再完善吧；
    现有功能包括小说搜索，增加书架、缓存、夜间模式、设置等。
    
## 项目

    框架：MVP+组件+ARouter
    网络框架：OKHTTP+Retrofit2+RxJava
    DB框架 :GreenDao
    图片框架：Glide
    XML解析框架：Jsoup
